A user must be able to:
- Create a task 
- Update a task
- Delete a task 
- Check off a task 
- Re-arrange tasks 
- Refresh the page and still see their tasks 


What to store in local storage
- The task's text content (e.g "Walk the dog")
- The checked state

Local storage example schema:

{
    task: "Walk the dog"
    checked: true
}

Approach:
1. Extract the id of the checkbox e.g "task-98-checkbox" ✅
2. Convert the id to a number e.g "task-98-checkbox" => 98  ✅
3. Find an object in local storage with an id of 98 ✅
4. Update the checkedState of that object to true ✅
